# ascii Simulation Game

- simulates the daily business in a coffee shop


- run Server using:

```sh
cd Server
npm install
npm start
``` 

- run client using:
cd Defold
open game.project in defold
- Project->Fetch Libraries

Controlls:
Arrow Keys - move
E - use, give, produce, ... 
<Tab> - Men√º
<1,2,3> - Join room 1, 2 or 3 while in menu
