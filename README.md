# ascii Simulation Game

- simulates the daily business in a coffee shop


- run Server using:

```sh
cd Server
npm install
npm start
``` 

- run client using:

  - `cd Defold`
  - open game.project in Defold
  - Project->Fetch Libraries

Controls:
- `Arrow Keys` - move
- `E` - use, give, produce, ... 
- `Tab` - Men√º
- `G` - enter room name while in menu
